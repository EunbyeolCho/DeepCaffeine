1. 코드 수정 순서

Mon: 자연

Wen: 채송

2. Basic Rule

앞선 코드를 수정할 때는 반드시 이전 코드를 지우지 않되 수정하는 사람의 이름과 함께 주석처리하여 수정한다.

코드를 지우는 것은 매주 일요일 정기회의때만 가능하다.
